# 跑步训练

每周跑三次：周二早上 easy run 5K，周四 intervals（公司楼下沿河），周日早上去 MacRitchie 蓄水池跑 long run，跑完在 Casuarina Curry 吃 prata 是固定仪式。

10K PB 是 56:30，2025年12月的 Standard Chartered Singapore Marathon 10K 组创的。2026年的目标是 sub-55，照现在的进度有点悬，主要是天气太热，配速上不去。

跑鞋是 Asics Gel-Nimbus 26，2026年1月买的，现在大概 380km，到 600km 换。手表是 Garmin Forerunner 265，Zone 2 的心率上限设在 152。

报了 2026年12月的 Standard Chartered 半马，第一次跑 21K。训练计划8月开始，照 Hal Higdon 的 Novice 2。
